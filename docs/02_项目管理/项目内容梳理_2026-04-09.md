## 📊 一、项目整体概况

### 1.1 项目定位
构建一套"实践主线贯穿 + 知识点按需融入"的空间转录组训练营公众号合集，覆盖 Visium/HD/Xenium/Stereo-seq 等全平台，以及 Scanpy/Squidpy/Cell2location 等全工具栈。

### 1.2 核心数据
- **总规划篇数**：206 篇
- **已完成篇数**：13 篇
- **完成率**：6.3%
- **系列分类**：S0-S12 共 13 个系列
- **发布节奏**：每周 3 篇（周一/周三/周五）
- **预计完成时间**：约 64 周（1.2 年）

### 1.3 项目结构
```
ST/
├── articles/          # 已发布文章（13篇）
├── notebooks/         # 配套 Jupyter Notebook（13篇）
├── docs/             # 项目文档
│   ├── 01_知识体系/  # 完整知识体系
│   ├── 02_项目管理/  # 排期表、索引
│   └── 03_规范/      # 术语表、质检清单
└── scripts/          # 自动化工具
    ├── harness.py    # 主控脚本
    ├── 01_lint_article.py
    ├── 02_article_to_notebook.py
    └── 03_sync_progress.py
```
