# 文章质检清单（LINT）

> 用途：发布前 5 分钟扫一遍，保证风格与可信性一致。

---

## A. 硬性禁止项（出现即改）
- [ ] 外链（http/https/DOI/期刊页面链接）
- [ ] 对文献主观评价（例如“写得踏实/组织得好/很全面/很权威”）
- [ ] 未定义缩写（首次出现没有“中文 + 英文/缩写 + 一句解释”）
- [ ] 教育口吻高密度（例如“你/先/必须/应该/别/记住/建议你”）

## B. 术语一致性
- [ ] 同一个概念全篇用词一致（例如“空间域/空间分区”不要混用）
- [ ] 英文/缩写只在首次定义处出现（后文尽量用中文）
- [ ] 专业词的比喻不改变含义（比喻只修辞，不替代定义）

## C. 证据链四件套（至少覆盖）
- [ ] 诊断图：质量/分布/伪影/批次（至少 1 张或 1 组）
- [ ] 对照：负对照/随机化/替代解释（至少 1 个）
- [ ] 敏感性：参数/方法替换后的稳定性（至少 1 个）
- [ ] 边界声明：能说明什么、不能说明什么（至少 1 段）

## D. 交付物与可复用
- [ ] 图表清单明确（读者知道会得到哪些图/表）
- [ ] 每节都有“这一节说清楚什么”
- [ ] 结尾有：常见坑（1–5条）+ 验收清单 + 下一篇预告

## E. 读者体验
- [ ] 开头 30 秒能读懂：这篇解决什么问题
- [ ] 段落不堆术语：术语首次出现就解释
- [ ] 语气像同伴复盘：少命令，多描述
