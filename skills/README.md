## 📁 Skills 列表

### st-article-writer

**用途**: 自动化生成空间转录组训练营文章

**版本**: 2.0.0

**核心功能**:
1. 从排期表读取文章定位
2. 遵循三段式结构生成文章
3. 自动运行 Linter 质检
4. 修复问题直到评分 ≥ 80

**输出**:
- articles/00X_标题.md - 文章正文（唯一交付物）

**质量保证**:
- ✅ Linter 评分 ≥ 80 分
- ✅ 证据链完整
- ✅ 术语一致
- ✅ 代码可执行

---

## 🎯 为什么放在项目目录下？

### 1. 项目特定性
- st-article-writer 是为 ST 项目量身定制的
- 依赖项目特定的规范、术语表、排期表
- 不适用于其他项目

### 2. 版本控制
- Skill 和项目代码一起版本管理
- 规范更新时，Skill 同步更新
- 便于追踪 Skill 的演进历史

### 3. 自包含性
- 项目目录包含所有运行所需的文件
- 不依赖全局环境（~/.claude/skills/）
- 便于项目迁移和分享
